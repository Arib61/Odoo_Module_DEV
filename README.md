# Module de Gestion des Projets Pédagogiques

## Description
Module Odoo complet pour la gestion des projets pédagogiques dans les établissements d'enseignement supérieur.

## Fonctionnalités
- **Gestion des étudiants** : Inscription, suivi académique, informations personnelles
- **Gestion des enseignants** : Profils, spécialités, projets supervisés
- **Gestion des projets** : Création, suivi, évaluation des projets pédagogiques
- **Gestion des tâches** : Planification et suivi des tâches des projets
- **Gestion des soutenances** : Planification des soutenances, jury, évaluation
- **Rapports et statistiques** : Tableaux de bord, analyses, exports PDF

## Installation
1. Placer le module dans le répertoire `addons` d'Odoo
2. Redémarrer le serveur Odoo
3. Aller dans Apps → Mettre à jour la liste des applications
4. Rechercher "Gestion des Projets Pédagogiques"
5. Installer le module

## Configuration
1. Créer des étudiants via le menu "Étudiants"
2. Créer des enseignants via le menu "Enseignants"
3. Créer des projets et les assigner aux étudiants et enseignants
4. Planifier les tâches et les soutenances

## Groupes de sécurité
- **Administrateur Projets** : Accès complet à toutes les fonctionnalités
- **Enseignant** : Gestion des projets supervisés, évaluation
- **Étudiant** : Consultation et gestion de ses propres projets
